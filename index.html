HTML

  • The head element contains information about the webpage
  • The body element represents the visibile content being shown to the user.
  • ul creates an unordered (bullet point) list, and li tag creates list items
  • ol creates an ordered list, with numbers for each point
  • h1 creates the largest heading and should only be used once per page
  • h2, h3, h4, h5, and h6 elements represent the level of heading a given text block represents
  • img tag contains information about images that are displayed on the webpage
  • img tag can contain many elements, the most important of which is the src element, which defines the location of the image
  • The p element represents a paragraph or block of text
  • br can be used to create a line of empty space, also known as a line break
  • The a tag creates links to the same webpage or other webpages
  • Within the a tag, the href attribute points to the URL for the link
  • Semantic HTML elements - the name of the element describes its content and the role it plays on the webpage
  • div and span are non-semantic HTML elements
  • div creates a division in the page
  • span creates inline division
  • Best practice is to use semantic elements whenever possible, especially when the content is visible to users

CSS

  • CSS is used to create style for the webpage
  • CSS determines the styles of a webpage in order to make a website visually appealing
  • CSS can also make a website simpler to use with a polished design and consistent organization
  • CSS can be added either inline, with an internal style sheet, or an external style sheet
  • Padding and Margin are often confused
  • Margin property indicates how much space we want around the outside of an element
  • Padding property adds space around the content inside an element
  • When we want to apply a rule to all elements visible on the page, we use a wild card symbol, or *
  • Two or more selectors can be combined into a single CSS rule using a comma
  • Applying style to certain elements, but not others, can be done by designating a class attribute as the selector

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: stage changes
  • git commit -m: add a commit message and commit changes
  • git pull origin main: pulls in the base branch to the feature branch
  • git push: pushes up changes to remote repository

JavaScript

  • JavaScript can be used to create interactivity on the webpage
  • JavaScript is a dynamic language with dynamic types - variables in JavaScriptare not directly associated with any particular value type, and any variable can be assigned and reassigned values of all types
  • In JavaScript, the keyword var can be used to declare a variable
  • Control flow is the order in which a comptuer executes the code in a script
  • In JavaScript, code runs from top to bottom
  • Conditional statements, such as if, allow us to interrupt control flow
  • An Array is a single variable that is used to hold a group of data
  • Arrays are typically used to hold data that is related in some way
  • Not all arrays must have multiple data types
  • A piece of data's location in an array is called an index
  • In JavaScript, indices always start at 0 and will always be whole numbers that increase by 1 each time
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an arrayby allowing a single code block to be executed over and over
  • A for loop contains three important statements: starting point, condition, and a final statement that allows the array to iterate over each item
  • A variable is a named container that allows us to store data in our code.
